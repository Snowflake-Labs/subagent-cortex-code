and custom).\n\n### Connection Management\n```bash\ncortex connections list\n```\n\nShows all configured Snowflake connections.\n\n### Search Operations\n```bash\ncortex search object <pattern>\ncortex search docs <query>\n```\n\nSearches Snowflake objects or documentation.\n\n## Event Stream Types\n\n### System Events\n```json\n{\n  \"type\": \"system\",\n  \"subtype\": \"type\": \"user\",\n  \"session_id\": \"...\",\n  \"message\": {\n    \"role\": \"user\",\n    \"content\": [\n      {\"type\": \"tool_result\", \"tool_use_id\": \"...\", \"content\": \"result or Write files\n- `edit` - Edit files\n- `glob` - File pattern matching\n- `grep` - Content search\n- `web_fetch` - Fetch web content\n- `ask_user_question` - Ask user questions\n- `task` - Task management\n- Plus skill-specific tools\n\n## Common Patterns\n\n### Simple Query\n```bash\ncortex -p \"Show top 10 customers\" \\\n  --output-format stream-json \\\n  --allowed-tools \"snowflake_sql_execute\" \"bash\"\n```\n\n### Data Quality Check\n```bash\ncortex -p \"Check data quality for SALES_DATA table\" \\\n  --output-format stream-json \\\n  --allowed-tools \"snowflake_sql_execute\" \"bash\" \"read\" \"write\"\n```\n\n### With Context Enrichment\n```bash\ncortex -p \"# Previous Context\nUser asked about customer segmentation.\n\n# Recent Cortex Work\nRan RFM analysis on customers table.\n\n# Current Request\nCreate a dynamic table for high-value customers\" \\\n  --output-format stream-json \\\n  --allowed-tools \"snowflake_sql_execute\" \"bash\" \"read\"\n```\n\n## Configuration Files\n\n### Settings Location\n`~/.snowflake/cortex/settings.json`\n\nKey settings:\n- `cortexAgentConnectionName` -
